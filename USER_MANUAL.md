## 1. Sign In and Access
- Open the app in your browser and sign in with your account.
- If you do not have an account yet, ask an Admin to create one in **System Config**.

## 2. Global Navigation
Use the left sidebar (admin) or bottom navigation (customer) to move between modules.

### Theme Toggle
- Switch between dark and light modes using the theme toggle in the header or settings area.
- The UI will remember your theme preference.

## 3. Admin Command Center
Admins have access to the enterprise dashboard modules:

### Financial Engine
- View KPIs, escrow, VAT/levies, logistics gross, and commissions.
- Use the **7D / 30D / 90D / All** selector to change the date range of the revenue trend.
- Debt and tax-safe metrics appear in the **Tax-Safe Engine** panel.
- `![Financial Engine chart](docs/screenshots/03-financial-engine.png)`

### Financial Cockpit
- See rollups, distribution status, and policy highlights.
- Review high-level cashflow and compliance signals.

### Dispatch Tower
- Monitor active orders and bay utilization.
- Bay capacity and hot thresholds are configured in **System Config**.
- Auto-Hot can be enabled to flag bays above the hot threshold.
- `![Dispatch Tower](docs/screenshots/04-dispatch-tower.png)`

### Fleet Command
- Track operational throughput, queue flow, and delivery rhythm.
- Use it for day-to-day dispatch health checks.

### QC Firewall
- Review audit trails and compliance events.
- Use filters to locate specific activities.

### Customer Data
- Inspect customer accounts and usage signals.
- Use search to find customers quickly.

### Vendor Network
- Track vendors, hubs, and operational health.

### Reseller Army
- Monitor reseller performance and team structures.

### Service Network (Pro)
- Review pro team coverage and availability.

### Finance Ledger
- View all ledger entries for revenue, escrow, adjustments, and refunds.
- Admins can add manual adjustments.
- Export the ledger to CSV for external reporting.
- `![Finance Ledger](docs/screenshots/05-finance-ledger.png)`

### System Config
- Manage users (create or update accounts).
- Configure dispatch bay capacity and hot thresholds.
- Toggle Auto-Hot if needed.
- `![System Config](docs/screenshots/09-system-config.png)`

## 4. Customer Experience
Customers interact through the customer portal:

### Storefront and Shopping
- Browse categories and items.
- Add items to the cart.
- `![Customer storefront](docs/screenshots/06-storefront.png)`

### Manage Cart
- Open **Manage Cart** to increase or decrease quantities.
- Cart changes are saved automatically.
- Save cart snapshots for later reuse and restore them when needed.
- `![Manage Cart](docs/screenshots/07-manage-cart.png)`

### Orders
- View recent and historical orders.
- Order details include status and totals.
- `![Orders list](docs/screenshots/08-orders.png)`

### Chats
- Message support or vendors.

### Locations
- Save and edit frequent delivery locations.

### Profile
- Update personal details and preferences.

## 5. Vendor Portal
Vendors can manage hub information and inventory.
- Edit hub details from the Vendor portal.
- Keep availability and status up to date.

## 6. Reseller Portal
Resellers can review brand performance and team alignment.
- Track brand status and campaign cadence.

## 7. Pro/Service Portal
Pros manage service portfolios and team coverage.
- Keep portfolio details current for accurate visibility.

## 8. Rider Portal
Riders see delivery assignments and status.
- Track assigned missions and completion details.

## 9. Common Tasks
### Create a User (Admin)
1. Go to **System Config**.
2. Select **Users**.
3. Fill in the user details and assign a role.
4. Save the account.
5. `![Create user modal](docs/screenshots/10-create-user.png)`

### Export Ledger (Admin)
1. Open **Finance Ledger**.
2. Choose optional filters.
3. Select **Export CSV**.
4. `![Export CSV button](docs/screenshots/11-export-ledger.png)`

### Save a Cart Snapshot (Customer)
1. Open **Manage Cart**.
2. Select **Save Snapshot**.
3. Give it a name and confirm.
4. `![Save snapshot](docs/screenshots/12-save-snapshot.png)`

## 10. Troubleshooting
- If a page does not load, refresh and try again.
- If data looks missing, confirm you are signed in and have the correct role.
- For persistent issues, contact your system administrator.
